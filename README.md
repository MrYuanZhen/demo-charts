# demo-charts

#### 启动说明

##### eureka启动
* 参数说明
   * service.nodePort #nodeport端口
   * image.repository #镜像地址
   * image.tag #镜像tag
  
***

##### demo-app启动
* 参数说明
   * service.nodePort #nodeport端口
   * image.repository #镜像地址
   * image.tag #镜像tag
   * eureka.eureka_url #注册eureka地址
   * eureka.eureka_port #注册eureka端口
   * app.demo_port #demo-app应用端口
